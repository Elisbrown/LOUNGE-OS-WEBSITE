Accessibility Statement

Last Updated: June 19, 2026

1. Commitment to Accessibility

LoungeOS is deeply committed to ensuring digital accessibility and user-friendly software interfaces for everyone, including individuals with visual, auditory, motor, or cognitive impairments. We continuously refine the LoungeOS POS client interface and website layouts to align with the Web Content Accessibility Guidelines (WCAG 2.1) Level AA requirements.

2. Web & App Accessibility Standards

Our digital platforms incorporate several features designed for high readability, easy control, and assistive technology compatibility:

  • Contrast Compliance: Text-to-background contrast ratios are audited to meet WCAG standards (minimum 4.5:1 ratio) to ensure ease of reading for low-vision operators.
  • Screen Reader Ready: We employ semantic HTML5 landmarks (such as <nav>, <main>, <header>, and <footer>) and detailed aria-label tags on interactive buttons, allowing screen reader software (e.g. VoiceOver, NVDA) to cleanly translate the site structure.
  • Scaleable Text: Font sizes on our platforms are fully responsive and scale accurately with system-level browser zoom without breaking layout containers.
  • Keyboard Navigation: All links, dialog triggers, and buttons can be selected and activated using a keyboard alone, with highly visible focus rings.

3. Assistive Technology Testing

We test our layouts across multiple viewport sizes and test compatibility with popular modern browsers (Safari, Chrome, Firefox) and built-in screen reader tools to ensure reliable navigation flows for users with accessibility requirements.

4. Feedback & Contact Support

If you encounter any layout bugs, low-contrast text, navigation blocks, or interactive areas that limit accessibility while using the LoungeOS website or client app, please reach out to us. We will work to resolve the barrier immediately. Contact us via WhatsApp at +237 679 690 703.
